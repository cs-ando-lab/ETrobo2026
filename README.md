# ET ロボコン 2026 走行体プログラムリポジトリ

本リポジトリは、ETロボコン2026で使用するコードとモデルを、チームで効率よく開発・管理するためのものです。

## 📦 開発環境 (Requirements)

開発を開始する前に、[公式wiki](https://github.com/ETrobocon/etrobo/wiki)を参考にして、必ず以下の環境を揃えてください。

| カテゴリ         |                         項目 | 推奨バージョン / 設定値                   |
| ---------------- | ---------------------------: | ----------------------------------------- |
| 仮想環境         |               WSL バージョン | WSL2                                      |
|                  | Linux ディストリビューション | Ubuntu 24.04 LTS                          |
| 開発エディタ     |                         本体 | Visual Studio Code                        |
|                  |                 必須拡張機能 | WSL 拡張機能(ms-vscode-remote.remote-wsl) |
| プラットフォーム |                     実行基盤 | SPIKE-RT                                  |

## 実行方法

実行方法には、通常の`make`コマンドによる実行とこのリポジトリ独自のスクリプトファイルによる実行の2種類の方法があります。

1. `make`コマンドによる実行

   ```bash
   cd ~/etrobo/
   make app=ETrobo2026/main up
   ```

2. スクリプトファイルによる実行

   ```bash
   cd ~/etrobo/workspace/ETrobo2026
   ./deploy.sh
   ```

   deploy.shによる実行では、「WSLと走行体の接続」「ビルド」「転送」が一括で行われます。
